# Dans Multi Pro Backend Tech Test

## Spesification
- Language: Javascript
- Framework: Node.js
- Database: MySQL
- Library: Express, MySQL, JWT, Axios, Joi, Jest, Winston

## Instalation Local

```
git clone https://github.com/ekowebdev/dans-test.git
cd dans-test
cp .env.example .env
npm install
npm start
```

## Instalation with Docker

```
git clone https://github.com/ekowebdev/dans-test.git
cd dans-test
cp .env.example .env
docker-compose up -d --build
```
